# CP Code Library

A curated collection of competitive programming templates, algorithms, and data structures in C++ — used in contests and practice.

## Contents

### Data Structures
- Segment Tree (with lazy propagation)
- Fenwick Tree / BIT
- Disjoint Set Union (DSU)
- Trie
- Sparse Table

### Graph Algorithms
- Dijkstra's Algorithm
- Bellman-Ford
- Floyd-Warshall
- Kruskal's MST
- BFS / DFS
- Topological Sort
- SCC (Kosaraju / Tarjan)
- LCA (Binary Lifting)

### Number Theory
- Sieve of Eratosthenes
- Prime Factorization
- Modular Arithmetic
- GCD / LCM (Euclidean)
- Extended Euclidean Algorithm
- Euler's Totient
- Matrix Exponentiation

### Dynamic Programming
- Knapsack (0/1, Unbounded)
- LIS / LCS
- Digit DP
- Bitmask DP
- DP on Trees

### String Algorithms
- KMP (Pattern Matching)
- Z-Algorithm
- Manacher's Algorithm
- Rolling Hash / Rabin-Karp
- Suffix Array

### Geometry
- Convex Hull (Graham Scan)
- Point / Line operations
- Polygon area and centroid

### Miscellaneous
- Binary Search / Ternary Search
- Two Pointers
- Sliding Window
- Meet in the Middle
- Fast I/O templates

## Usage

Each file is self-contained. Include the relevant header or copy the template into your solution.

```cpp
#include "data-structures/segtree.cpp"
```
